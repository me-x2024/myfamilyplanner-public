2. Welche Daten werden gespeichert?

myFamilyPlanner speichert ausschließlich die Daten, die du selbst einträgst:

  • Einkaufsliste (Artikel, Kategorien)
  • Wochenplan (Termine, Datum, Uhrzeit, Familienmitglied)
  • Rezepte (Name, Zutaten, Zubereitung)
  • Familienmitglieder (Namen)
  • Raum-Code und Passwort (verschlüsselt als Hash gespeichert)

Es werden keine Standortdaten, Gerätedaten oder persönliche Profile erstellt.

3. Wo werden die Daten gespeichert?

Die Daten werden in Supabase (EU-Region) gespeichert. Supabase ist DSGVO-konform. Alle Verbindungen laufen verschlüsselt über HTTPS.

Auf deinem Gerät werden Daten zusätzlich im lokalen Gerätespeicher (localStorage) zwischengespeichert für schnellen Zugriff ohne Internetverbindung.

4. KI-Assistent (ChatGPT-Integration)

Der optionale FamilyPlanner-GPT basiert auf ChatGPT von OpenAI. Wenn du den KI-Assistenten verwendest, werden deine Sprach- oder Texteingaben an OpenAI übermittelt. Die Verarbeitung durch OpenAI unterliegt OpenAIs Datenschutzrichtlinien.

Dein Agent-Token wird nur im Gesprächsverlauf gespeichert und nicht dauerhaft von OpenAI hinterlegt. Er wird ausschließlich zur Authentifizierung gegenüber der FamilyPlanner-API verwendet.

5. Weitergabe an Dritte

Deine Familiendaten werden nicht an Dritte verkauft oder weitergegeben. Ausnahme: technisch notwendige Dienstleister (Supabase für Datenspeicherung, OpenAI für die optionale KI-Funktion).

6. Löschung deiner Daten

Du kannst deinen Familienraum und alle dazugehörigen Daten jederzeit in der App unter Einstellungen → Raum löschen dauerhaft entfernen. Alle Daten werden dann vollständig und unwiderruflich gelöscht.

Die App verwendet keine Tracking-Cookies. Der lokale Gerätespeicher (localStorage) dient ausschließlich zur Funktionssicherung der App.
